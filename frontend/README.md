# 🐾 Clínica Veterinaria SAVIA - Frontend

Aplicación web para gestión de clínica veterinaria y centro de adopción.

## 🚀 Inicio Rápido

docker-compose build
docker-compose up

## 🔑 Credenciales de Prueba

- **Admin**: admin@savia.com / 1234
- **Vet**: vet@savia.com / 1234
- **User**: user@gmail.com / 1234


## 📁 Estructura

```
src/
├── views/          # Páginas
├── components/     # Componentes reutilizables
├── stores/         # Estado global (Pinia)
├── router/         # Rutas
├── api/            # Configuración Axios
└── models/         # Tipos TypeScript
```

## ✨ Funcionalidades

- ✅ Autenticación JWT
- ✅ CRUD Mascotas
- ✅ CRUD Usuarios
- ✅ Panel de Administración
- ✅ Búsqueda y filtros
- ✅ Validación de formularios
- ✅ Roles de usuario

## 📝 Notas

- Backend debe estar en `http://localhost:8407`
- Todo el código está comentado en español
- Sin errores de TypeScript
